# Grupo Navegante - Website Institucional

Este é o código-fonte do site institucional do Grupo Navegante, uma empresa especializada em soluções de logística empresarial e produtos inteligentes.

O projeto foi desenvolvido como um site estático, focado em alta fidelidade visual, responsividade e código limpo.

## Tecnologias Utilizadas

- HTML5 (Estrutura semântica)
- CSS3 (Flexbox, Grid Layout, Animações e Responsividade)
- JavaScript (Manipulação do DOM e envio de formulário)
- FontAwesome (Ícones)
- Google Fonts (Tipografias Montserrat e Open Sans)
- FormSubmit (Integração de formulário de contato via API)

## Estrutura do Projeto

O projeto segue uma organização de diretórios para facilitar a manutenção:

- / (Raiz): Contém todos os arquivos HTML (index.html, servicos.html, historia.html, etc).
- /assets/css: Contém as folhas de estilo separadas por página.
- /assets/js: Contém os scripts de interatividade e lógica do formulário.
- /assets/img: Contém todas as imagens, logotipos e ícones utilizados.

## Funcionalidades

1. Navegação Responsiva: Menu adaptável para desktops e dispositivos móveis.
2. Animações ao Rolar (Scroll Reveal): Elementos aparecem suavemente conforme o usuário navega.
3. Contadores Dinâmicos: Números na seção de métricas contam automaticamente.
4. Formulário de Contato: Integração funcional para envio de e-mails direto da página.
5. Linha do Tempo: Layout interativo na página Nossa História.

## Como Executar

Para visualizar o projeto:

1. Baixe os arquivos deste repositório.
2. Certifique-se de manter a estrutura de pastas (assets) correta.
3. Abra o arquivo "index.html" em qualquer navegador moderno (Chrome, Firefox, Edge).

Não é necessário instalar servidores ou dependências adicionais para visualizar o layout. Para que o formulário de contato funcione, é necessário estar conectado à internet.
